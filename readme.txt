Aseba packaging script for Windows

These scripts are currently not part of the Aseba source code. For more information
on Aseba, please visit:
	http://aseba.wikidot.com
	
The source code of Aseba can be found here:
	https://github.com/aseba-community/aseba


To use these scripts, you will first need a working installation of NSIS, compiled
with the advanced logging support. See http://nsis.sourceforge.net/Main_Page. For
special build, refer to: http://nsis.sourceforge.net/Special_Builds.

You will also need Aseba binaries, as well as Qt's binaries (http://qt.nokia.com/products/).
